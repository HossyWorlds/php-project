# PHP(laravel)を用いたプロジェクト

## レポジトリの作り方とファイルへの接続法

### コマンド

まず初めにレポジトリを GitHub で作る。  
そしたら以下のコマンドを実行する。  
`git init`  
"README.md"を作った後に、
`git add README.md`  
`git commit -m "first commit"`  
`git push -u origin main` (なんか`-u`入れなくてもできちゃった...)  
そしたら GitHub 上のレポジトリのページの画面が変わる。
